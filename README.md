# CarClassification

### What Card Dataset includes:
- train
- test
- valid

Each folder has 53 subfolders for each type of card, Joker included.
- Ace of Spades
- Two of Spades
- Three of Spades
...

### Models Made
We have two models:
- One uses CNN and then two normal feed-forward layers
- The other just uses feed-forward layers
